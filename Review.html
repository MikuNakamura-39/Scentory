Stories

香りを選んだ時間も、完成したあとも。

Scentoryで生まれた体験のストーリーをご覧ください。

★★★★★　Nさん、Mさん / 20代 / 社会人

恋人と、忘れられない時間に。

記念日のデートで利用しました。香水を自分たちで作るのは初めてでしたが、スタッフの方が丁寧に説明してくれるので安心して楽しめました。香りを一緒に選ぶ時間もすごく楽しくて、2人の思い出になりました。完成した香水を持ち帰れるのも嬉しくて、帰ってからもその香りを嗅ぐと今日のことを思い出します。普通のデートとは違う特別な体験ができました。

★★★★★　Tさん、Cさん / 20代 / 学生

旅の思い出を、香りにして持ち帰る。

旅行で浅草に来たときに体験しました。お店の雰囲気がおしゃれで、香りもたくさんあって選ぶのがとても楽しかったです。自分の好みを考えながら香水を作るのは初めてで、友達と「この香りいいね」と話しながら作る時間がすごく盛り上がりました。完成したボトルも可愛くて写真映えします。旅行の思い出としてとても良い体験になりました。

★★★★★　Aさん / 30代 / 社会人

はじめての調香体験でも安心でした。

普段あまり香水を使わないのですが、自分に合う香りを作れると聞いて興味を持ち参加しました。香りの説明や組み合わせのアドバイスをもらいながら作れるので、初心者でも分かりやすかったです。自分で作った香りは愛着が湧きますし、普段使いもしやすい香りになりました。1人でも気軽に参加できて、とても良い体験でした。

★★★★★　Yさん / 20代 / 社会人

静かな空間で、気持ちまで整いました。

慌ただしい日々の中で少し立ち止まりたくて来店しました。香りをひとつずつ比べていく時間が想像以上に心地よく、自然と気持ちが落ち着いていくのを感じました。完成した香りも普段使いしやすく、それでいて自分だけの特別感があって、毎日の気分をやわらかく整えてくれます。

★★★★★　Kさん / 30代 / 会社員

贈り物にも、自分時間にも良い体験でした。

プレゼント目的で予約しましたが、香りを選ぶ工程そのものがとても楽しくて、自分自身のための時間としてもしっかり満たされました。知識がなくても安心して選べたのがよかったです。特別感のある体験になりそうで、記念日やお祝いにもぴったりだと思いました。仕上がりにもきちんと満足しています。

★★★★★　Rさん / 20代 / 学生

香りを選ぶ時間そのものが思い出になる。

友人同士で参加しましたが、どれを選ぶか話しながら進める時間がとても楽しかったです。完成品だけではなく、選ぶ過程まで含めてしっかり記憶に残る体験でした。一人ひとり違う香りになっても納得感があり、完成後に並べて比べる時間もすごく盛り上がりました。

★★★★★　Mさん / 40代 / 自営業

香りを整える時間が心地よかったです。

最初は直感だけで決めるのかと思っていましたが、香りの重なりや印象の変化を見ながら少しずつ整えていけるので、とても納得感がありました。完成した香りにも落ち着いた品が出ました。落ち着いた空間の中で、自分のための時間を持てたこと自体が印象的でした。

★★★★★　Sさん / 30代 / 社会人

忙しい日にもそっと寄り添う香り。

普段は香りに詳しいほうではありませんが、説明がわかりやすく、自分の感覚を大事にしながら選べたのがよかったです。完成した一本は日常に自然になじんでくれました。自宅で使うたびに、その場で香りを選んだ時間まで一緒に思い出されて、気持ちがやわらぎます。

★★★★★　Hさん / 20代 / 会社員

新しい自分の好みに出会えました。

普段は選ばない系統の香りも試してみたことで、意外と自分に合うものが見えてきました。少しずつ試しながら好みを見つけていく時間がとても新鮮でした。完成した香りには特別感もありつつ、しっかり自分らしさがあって、気持ちまで前向きになるような仕上がりでした。
